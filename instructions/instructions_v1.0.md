# Goal
The goal of this project is to make a csv editor

# Requirements
- React web app. The user launches with npm start from terminal.
- left column is a local file browser
- when user double cliks on csv, its content displays on the main window
- the file should be editable like in excel, where columns and rows are separated by lines. If user double clicks, the cell should be editable.
- there should be a save button, new file button, save as button

# Output
- App code in folder ./app
